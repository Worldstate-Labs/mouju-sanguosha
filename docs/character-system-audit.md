## 本轮发现并修复

1. 【过河拆桥/顺手牵羊】目标可能在无懈链中用掉最后一张牌，效果恢复时仍进入选区，导致 legalActions 为空并永久死锁。复现 seed：50089。
2. 判定牌未先公开便询问【鬼才】，玩家无法依据自然判定决定是否改判。
3. 响应阶段多张同名【闪/杀】被渲染成相同按钮，无法绑定具体实体牌。
4. 转换技能只公开最终牌名，不公开【龙胆/武圣/倾国/急救】等技能来源和实体牌。
5. 【过河拆桥】弃置隐藏手牌后仍隐藏弃置结果；进入公开弃牌堆后应揭示。
6. 普通弃牌及制衡、结姻、青囊、离间、刚烈、流离、雌雄双股剑、寒冰剑、贯石斧等成本牌进入弃牌堆后没有公开具体牌。
7. 【反馈】取得公开装备时仍显示成未知牌，缺少定向转移动画元数据。
8. 护驾、激将、借刀杀人和青龙偃月刀追加杀没有公开提供者使用的实体牌及转换技能来源。
9. 对手手牌数量和体力虽然由服务端返回，但在窄席位中会被 Agent 状态和武将信息挤出；现固定在席位底部并显示明确文字。
10. 对局结束页和已开始房间的“创建新房间”只修改 URL，不清空客户端房间状态，点击后看似无反应。

每个修复均有确定性测试或混沌 seed 回归。版本 38 已公开发布；本文件后续新增的并发与强化不变量门禁将在下一次生产检查点随源码发布。
